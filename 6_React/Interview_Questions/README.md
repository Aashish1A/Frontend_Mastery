# React Interview Questions

4. **What are React components?**

6. **What are props in React?**
7. **What is state in React?**
8. **What are React hooks?**
9. **What is the useState hook?**
10. **What is the useEffect hook?**

18. **What is code splitting in React?**

## Interview Tips:

- Always explain your thought process
- Start with a simple solution, then optimize
- Discuss trade-offs and alternatives
- Be prepared to write code
- Know React DevTools usage
- Understand React's philosophy and principles
- Stay updated with latest React features
- Practice coding challenges regularly
