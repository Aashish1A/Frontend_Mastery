1. What is Git?

Answer:
Git is a distributed version control system used to track changes in code.

2. What is GitHub?

Answer:
GitHub is a cloud platform for hosting Git repositories and collaborating with developers.

3. Difference between Git and GitHub
Git	GitHub
Version control tool	Hosting platform

4. What is a repository?

A repository (repo) is a project folder containing code and version history.

5. What is git clone?
git clone repository-url

Copies a remote repository to local machine.

6. What is git pull?
git pull

Fetches and merges changes from remote repository.

7. What is git push?
git push origin main

Uploads local changes to GitHub.

8. What is branching?

Branching allows developers to work on features separately.

git branch feature-login
git checkout feature-login
9. What is a pull request?

Pull Request (PR) is a request to merge code from one branch to another.

10. What is git merge?
git merge branch-name

Combines changes from one branch into another.
